## Amazon VPC(Virtual Private Cloud)

1. AWS 콘솔에서 VPC 대시보드에 접속
2. VPC 생성
   - VPC 명, CIDR 등..
3. 서브넷 설정
   - 서브넷 명, 대상 VPC, 가용 영역
4. 인터넷 연결
   - 인터넷 게이트웨이 생성 및 설정
   - 라우팅테이블에 인터넷 게이트웨이 추가
